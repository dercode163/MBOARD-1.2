# MBOARD

MBOARD is a compact 9-key macropad powered by an RP2040 microcontroller perfect for custom shortcuts,creative input controls.

#  Features

- 9 programmable keys 
- RP2040 MCU 
- Easy to customize firmware and layout  



# Bill of Materials (BOM)
List of all components used in this project:

| Part | Quantity | Description | 
|------|----------|-------------|
| Mechanical Key Switches | 9 | MX-style switches |
| Keycaps | 9 | Compatible with MX switches | 
| RP2040 Microcontroller | 1 | e.g., Seeed XIAO RP2040 | 
| Custom PCB | 1 | m board Design | 
| 3D Printed Case | 1 | Base + Cover + Top plate |
